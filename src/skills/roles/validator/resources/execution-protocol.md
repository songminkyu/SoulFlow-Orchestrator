# Validator Execution Protocol

## 1. 빌드 검증

- 프로젝트 언어에 맞는 빌드 명령 실행 (`_shared/lang/` 참조)
- 에러 0건 확인
- 에러 발생 시 목록 + 관련 파일 보고

## 2. 테스트 검증

- 프로젝트 언어에 맞는 테스트 명령 실행 (`_shared/lang/` 참조)
- 전체 통과 확인
- 실패 시 실패 테스트명 + 에러 메시지 보고

## 3. Lint 검증

- 프로젝트 언어에 맞는 린트 명령 실행 (`_shared/lang/` 참조)
- 에러 0건 확인
- 워닝은 보고만 (차단 안 함)

## 4. 결과 보고

```
**빌드**: PASS/FAIL (에러 N건)
**테스트**: PASS/FAIL (N/M 통과)
**lint**: PASS/FAIL (에러 N건, 워닝 M건)
**결론**: PASS / FAIL
```

## 금지 사항

- 코드 직접 수정 (결과만 보고)
- 테스트 스킵/무시
