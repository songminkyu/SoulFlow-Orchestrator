# Cron 표현식 레퍼런스

## 형식

```
분(0-59) 시(0-23) 일(1-31) 월(1-12) 요일(0-7, 0=일요일)
```

## 자주 쓰는 패턴

| 표현식 | 의미 |
|--------|------|
| `0 9 * * *` | 매일 오전 9시 |
| `0 9 * * 1-5` | 평일(월~금) 오전 9시 |
| `0 9 * * 1` | 매주 월요일 오전 9시 |
| `0 9,18 * * *` | 매일 오전 9시, 오후 6시 |
| `0 */2 * * *` | 2시간마다 |
| `*/30 * * * *` | 30분마다 |
| `0 0 1 * *` | 매월 1일 자정 |
| `0 0 * * 0` | 매주 일요일 자정 |
| `0 0 1 1 *` | 매년 1월 1일 자정 |

## every_seconds 패턴

| 표현식 | 의미 |
|--------|------|
| `every_seconds: 60` | 1분마다 |
| `every_seconds: 300` | 5분마다 |
| `every_seconds: 1800` | 30분마다 |
| `every_seconds: 3600` | 1시간마다 |

## 주요 타임존

| 지역 | IANA 타임존 |
|------|------------|
| 서울 | `Asia/Seoul` |
| 도쿄 | `Asia/Tokyo` |
| 뉴욕 | `America/New_York` |
| 런던 | `Europe/London` |
| UTC | `UTC` |
| 밴쿠버 | `America/Vancouver` |
| LA | `America/Los_Angeles` |

## 자연어 → cron 변환 예시

| 사용자 발화 | 파라미터 |
|------------|---------|
| "매일 아침 8시에" | `cron_expr: "0 8 * * *", tz: "Asia/Seoul"` |
| "30분 후에" | `at: "<now + 30min ISO>"` |
| "매주 월요일 오전에" | `cron_expr: "0 9 * * 1", tz: "Asia/Seoul"` |
| "매시간" | `every_seconds: 3600` |
| "5분마다" | `every_seconds: 300` |
