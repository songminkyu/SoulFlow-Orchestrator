# Session Metrics — Clarification Debt (CD)

세션 품질을 측정하는 CD 점수 시스템.

## CD 이벤트

| 지표 | 점수 | 설명 |
|------|------|------|
| clarify | +10 | 에이전트가 사용자에게 질문 |
| correct | +25 | 동일 도구 3+회 에러 후 방향 전환 |
| redo | +40 | 이전 작업 롤백 후 재시도 |

## 임계값

| CD 점수 | 상태 | 행동 |
|---------|------|------|
| 0-15 | 건강 | 정상 진행 |
| 16-49 | 주의 | 현 접근 재검토 |
| 50-79 | 경고 | RCA 기록 필수 |
| 80+ | 위험 | 세션 일시정지, 사용자 확인 |

## 건강한 세션

- CD ≤ 10-15가 목표
- clarify 1-2회는 정상 (불확실성 해소)
- correct/redo 없음이 이상적
